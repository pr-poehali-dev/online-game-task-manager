## Важно понимать про архитектуру
Frontend (то, что видит пользователь в браузере) и backend (Python-функции, база данных, файловое хранилище) — это разные части. Backend продолжит работать на инфраструктуре текущей платформы (адреса вида `functions.poehali.dev/...` в файле `backend/func2url.json`) — это нормально и трогать их не нужно, менять их вручную нельзя, иначе сайт перестанет работать.

**На ваш сервер forge.la2era.com нужно перенести только собранный frontend** (папку `dist` после сборки).

## Шаг 1. Скачайте актуальный код или билд
В интерфейсе платформы: **Скачать → Скачать билд** — вы получите готовую папку с HTML/JS/CSS, которую можно сразу выложить на сервер.

Если нужен исходный код (например, чтобы дособрать самостоятельно) — **Скачать → Скачать код**, либо подключите GitHub через **Скачать → Подключить GitHub**.

## Шаг 2. Разместите файлы на forge.la2era.com
1. Загрузите содержимое папки `dist` в корень веб-сервера, который обслуживает домен `forge.la2era.com`
2. Убедитесь, что на сервере настроен SSL-сертификат (https)
3. Настройте сервер так, чтобы все несуществующие пути отдавали `index.html` (SPA-роутинг). Пример для nginx:
   ```nginx
   location / {
       try_files $uri /index.html;
   }
   ```

## Шаг 3. Обновите домен в настройках проекта (обязательно)
В проекте есть секрет **APP_URL** — это адрес, который backend подставляет в кнопки Telegram-уведомлений («Открыть задачу», «Мои задачи» и т.д.). Сейчас там указан старый адрес превью.

Зайдите в раздел секретов проекта и измените значение **APP_URL** на:
```
https://forge.la2era.com
```
Без слэша в конце.

## Шаг 4. Проверьте Telegram-бота
Бот (webhook) работает независимо от домена сайта — он привязан к облачной функции `tg-webhook` и продолжит работать без изменений. Дополнительных действий с ботом не требуется.

## Шаг 5. Проверка после переноса
- Откройте `https://forge.la2era.com` — должна загрузиться страница входа
- Войдите через Telegram-бота
- Откройте любую задачу и назначьте себя исполнителем — должно прийти уведомление в Telegram с рабочей ссылкой на `forge.la2era.com`
- Проверьте `https://forge.la2era.com/robots.txt` — там должно быть `Disallow: /`, чтобы сайт не индексировался поисковиками

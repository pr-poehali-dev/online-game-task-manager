# Обновление — раздел «К рестарту» разбит по серверам

## Что изменилось

В разделе «К рестарту» задачи теперь группируются в отдельные блоки по серверам
(С4х1 / HFx3 old / HF new и т.д.) — вместо одного общего списка. Внутри каждого блока
задачи отображаются как раньше (карточка с приоритетом, статусом деплоя, исполнителями).
Серверы без задач к рестарту не показываются.

Логика "Готово"/"В архив" не менялась — работает как прежде, просто внутри своей группы.

## Файл для переноса

```
src/pages/index/Restart.tsx
```

Это единственный изменённый файл — просто замените его целиком на вашем сервере.
Backend и база данных не затрагивались, миграции не требуются.

## После переноса

```bash
npm run build
```

Backend перезапускать не нужно.

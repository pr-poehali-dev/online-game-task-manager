# Обновление — кнопка «Готово» в разделе «К рестарту» стала нейтральной

## Что изменилось

Кнопка «Готово» для невыполненных задач раньше была зелёной — из-за этого визуально
казалось, что задача уже отмечена выполненной, хотя её ещё нужно было нажать.

Теперь кнопка нейтральная (серая, как обычное действие) и подсвечивается зелёным
только при наведении курсора — так понятнее, что это кликабельное действие,
а не статус. Зелёная подсветка самой карточки после выполнения не менялась.

## Файл для переноса

```
src/pages/index/Restart.tsx
```

Единственный изменённый файл. Backend и база данных не затрагивались.

## После переноса

```bash
npm run build
```
